# Lesser Restoration

Abjuration 2nd lv

|              |               |
| ------------ | ------------- |
| casting time | 1 action      |
| range        | Touch         |
| components   | V,S           |
| duration     | Instantaneous |

##

You touch a creature and can end either one disease or one condition afflicting it.  
The condition can be:

- blinded
- deafened
- paralyzed
- poisoned
